# Asteroid

Shoot the asteroid, don't get hit!
Simple asteroid game made with python, pygame, and bootdev.


### Game info
The game last about 3 minutes, with low difficulty.
The spaceship reuse material collected from asteroids you hit, or something like that.
The more you break, the faster your ship, incrementally.

The incremental part is not calibrated past level 5, when the lvl bar turn green, everything goes BRUUUH 


### Controls
ZQSD - space to shoot
